### SeaBattle3D

### Requirements:

* JDK 8

### Run application:

```.\gradlew clean run```

### Run tests:

```.\gradlew clean test```

### View reports:

Default ```build/reports/tests/test/index.html```  
Allure ```build/reports/allure-report/allureReport/index.html```   
Jacoco ```build/reports/jacoco/test/html/index.html```
